About

QShr.tn is an open-source, anonymous URL shortening service with a free to use API for everybody.

As we offer a completely anonymous service, the only information we store on a link is the URL on our page and the redirect URL. The creation time, creator and any people accessing the links are never stored.

This site is primarily developed by me, Froogo, if you have any suggestions or bugs you can report them to me at my email: harry@froogo.co.uk.
